# 电信运营商业务知识图谱（交互网页）

本仓库提供一个纯前端的知识图谱演示页面：`index.html`。

## 功能
- 实体节点可拖拽移动（客户域、产品域、网络域、计费域、运营域）。
- 关系线与关系标签实时跟随节点位置更新。
- 点击节点可高亮上下游关系并显示关系明细。
- 可按业务域筛选展示。

## 使用方式
直接在浏览器打开 `index.html`，或运行本地静态服务：

```bash
python3 -m http.server 8000
```

然后访问：`http://localhost:8000`

## 说明
图谱内容按电信运营商常见业务数据对象进行建模，用于演示型可视化与业务梳理讨论。
